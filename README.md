# Screenshots

Add the following screenshots to this directory:

## Required
- `homepage-pt.png` — Homepage in Portuguese
- `homepage-en.png` — Homepage in English  
- `homepage-es.png` — Homepage in Spanish
- `ai-overview.png` — Screenshot of site appearing in Google AI Overviews
- `cloudfront-distribution.png` — CloudFront distribution config (sensitive values blurred)
- `lightsail-instance.png` — Lightsail instance dashboard

## Optional but recommended
- `gsc-performance.png` — Google Search Console performance (last 3 months)
- `polylang-config.png` — Polylang language configuration
- `s3-backup-bucket.png` — S3 bucket structure showing backup files

## How to capture AI Overview screenshot
1. Open Chrome in incognito mode
2. Search: "loft para alugar no Rio de Janeiro"
3. Screenshot the AI Overview panel at the top
4. Crop to show the mention of loftgigoiarj.com.br
